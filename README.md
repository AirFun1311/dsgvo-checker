# DSGVO/GDPR Compliance Checker

Automated compliance checking tool for German SMEs and healthcare providers.

## Features

### 1. Website Scanner
- Privacy policy compliance check
- Cookie consent verification
- HTTPS enforcement check
- Data collection transparency

### 2. Document Analyzer
- Privacy policy gap analysis
- Data processing agreement templates
- Record of processing activities (RoPA) generator
- Data breach notification templates

### 3. Compliance Reporting
- Automated audit reports
- Risk assessment scoring
- Remediation recommendations
- Executive summary generation

## Compliance Standards

- **DSGVO (German)** - Datenschutz-Grundverordnung
- **GDPR (EU)** - General Data Protection Regulation  
- **BDSG** - Bundesdatenschutzgesetz
- **TTDSG** - Telekommunikation-Telemedien-Datenschutzgesetz
- **ePrivacy** - Cookie and electronic communication regulation

## Report Sections

1. **Executive Summary** - Key findings and risk level
2. **Technical Assessment** - Website security and configuration
3. **Policy Compliance** - Privacy policy gap analysis
4. **Cookie Audit** - Cookie consent implementation
5. **Data Processing** - Data collection and sharing practices
6. **Remediation Plan** - Step-by-step fixes with priorities
